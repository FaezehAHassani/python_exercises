_ex1:_

my_list = [1, 2, 3, 4]

my_list[0] **returns 1**

my_list[-1] and my_list[3] **returns 4**

print(my_list[0:3]) **returns [1, 2, 3]**

print(my_list[:-3]) **returns [1]**

my_list[0] = 0
print(my_list) **returns [0, 2, 3, 4]**
